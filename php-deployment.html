Words of Havoc - PHP Deployment Guide

This guide explains how to deploy Words of Havoc with a shared leaderboard using basic PHP hosting.

Simple Deployment Steps

  1. Ensure your web hosting supports PHP (most shared hosting services do)
  2. Edit game.js and set LEADERBOARD_SERVER_ENABLED = true;
  3. Upload all the game files to your hosting service, including leaderboard.php
  4. Make sure leaderboard-data.json file is writable by the server: 
    chmod 666 leaderboard-data.json
    (If this file doesn't exist yet, it will be created automatically but the directory must be writable)
  5. Access your game by navigating to the index.html file on your server
Note: Your PHP hosting service needs to allow writing to files for the leaderboard to work. Most shared hosting services allow this by default.

Files You Need to Upload

  • index.html - Main game HTML
  • game.js - Game logic (with LEADERBOARD_SERVER_ENABLED = true;)
  • word-pairs.js - Word data for the game
  • styles.css - Game styling
  • leaderboard.php - PHP script for the leaderboard
  • Any other asset files (sounds, images, etc.)

Testing

After deployment, test the leaderboard by:

  1. Playing a complete game
  2. Submitting a score with your name
  3. Checking if your score appears on the leaderboard
  4. Verifying that a file called leaderboard-data.json was created on your server

Troubleshooting

  • If the leaderboard doesn't save scores, check file permissions for leaderboard-data.json
  • If you see "Failed to fetch" errors in the console, make sure PHP is properly configured on your server
  • Check server error logs for any PHP-related errors

That's it! Enjoy your game with a shared leaderboard without needing Node.js.
